PROJECT TITLE: Clocks
PURPOSE OF PROJECT: Maintain clocks, run and print them, and use multithreading
VERSION: 1.0
HOW TO START THIS PROJECT: 
    there are 3 modes to start this project:
        - running a single clock: use the "main" method in "MyClock" class
        - running three clocks (sequentially): use the "main" method in "MyMultiClock" class
        - running three clocks (in parallel): use the "main" method in "MyMultiThreadedClock" class
